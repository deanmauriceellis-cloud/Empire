DESIGN DECISIONS:
- No @pixi/tilemap (not v8-compatible) — sprite pooling with frustum culling instead
- RenderableState abstraction — renderer never touches GameState directly; bridge adapts
- Graphics-to-Texture for placeholders — draw once, use Sprite for all instances
- Fog as overlay sprites — separate layer, not modifying tile textures
- Lerp-based animation — no tweening library needed
- Pool sizing up to MAP_WIDTH × MAP_HEIGHT (6000) sprites max
