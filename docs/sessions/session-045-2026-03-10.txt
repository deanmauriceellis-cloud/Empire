ARCHITECTURE NOTES
------------------
Building construction follows same pattern as city production:
  1. Player issues build action → resources consumed immediately
  2. BuildingState created with work=0, constructorId set
  3. Each turn, tickBuildingConstruction advances work by 1
  4. Constructor must remain at building location (verified each tick)
  5. On completion: building.complete=true, deposit.buildingComplete=true (if deposit), constructor consumed via killUnit()

Tech research is tracked but thresholds/unlocks/bonuses are Phase 6.
AI construction management is Phase 8.
Building capture/destruction mechanics are implied by the data model but not yet implemented.
